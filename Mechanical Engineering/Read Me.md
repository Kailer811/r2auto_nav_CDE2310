# Mechanical Design Files for CDE2310 AMR Project

This folder contains the mechanical design documentation and fabrication files for CDE2310 By group 8. The files here support the design, fabrication, assembly, and review of the payload storage, feeding, launcher, and supporting structural components.

## Folder Structure

### 1. CAD
Contains the CAD source files for the mechanical design, including design iterations and the latest geometry used for the project.

### 2. 3D Printed Parts Files
Contains print-ready files for all 3D printed components used in the final design. 

### 3. Screenshots
Contains screenshots of CAD models, subsystem layouts, and other visual references used for documentation, presentations, or design discussion.

### 4. Assembly Drawing
Contains assembly drawings and dimensioned references showing how the parts are arranged and fitted together.

### 5. BOM
Contains the Bill of Materials (BOM), including part names, quantities, and relevant notes such as accurate part locations or centre-of-gravity related details where applicable.


## Latest Version
The latest approved design files should be able to be identified by V4. Older files, if present, are retained only for reference and comparison.
